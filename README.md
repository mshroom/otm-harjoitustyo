# Ohjelmistotekniikan menetelmät -harjoitustyö

Harjoitustyön aiheena on sovellus, johon käyttäjät voivat kirjata tietoa liikuntasuorituksistaan, asettaa itselleen tavoitteita ja seurata niiden toteutumista.
